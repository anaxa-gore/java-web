# Répertoire TMP

Ce répertoire peut être utilisé lors de la génération via Gulp pour placer des fichier temporaires.

Il est possible :
- d'ajouter des sous-répertoires à volonté,
- d'ajouter/modifier/supprimer les fichiers contenus dans ce répertoire tout au long du processus de
build

Il est conseillé :
- de supprimer le contenu de ce répertoire AVANT chaque build !
