# Répertoire TEST

Ce répertoire est destiné à contenir les tests unitaires, à lancer avec Gulp.

La structure doit être identique à celle du répertoire `src/`. Par exemple :
```
+-- src/
|  +-- module1/
|  |  +-- mod1.js
|  |  +-- mod1.css
|  |  +-- mod1.template.html
|  +-- module2/
|  |  +-- mod2.js
|  |  +-- mod2.css
|  |  +-- mod2.template.html
```

```
+-- test/
|  +-- module1/
|  |  +-- mod1.specs.js
|  +-- module2/
|  |  +-- mod2.specs.js
```
